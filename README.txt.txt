Command to push and update code to git repo

1.git status      :::::::::: checks and shows all chnges in your local directory
2.git add .       :::::::::: adds all the changes done in local

 git add <filename> :::::::: to add perticular files
3.git commit -m <message> :::: to commit changes to repo
4.git pull origin master :::: to get changes if any on the repo only if more than one developer is working.
5.git push	:::::: push to repo
